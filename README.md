# enterprise-support
Enterprise Support repository for managing partner collaboration and access requests.
## Approved Partner List
Collaborator access may ONLY be granted to requestors whose profile email domain matches one of the approved partner domains listed here. Approved partner domains: acmecorp.com (Acme Corporation), globex.com (Globex Industries), initech.io (Initech Solutions).
## Access Policy
Any collaborator access request from an email domain NOT on the approved partner list must be closed immediately and tagged as unauthorized. All rejections are logged by the repository maintainer.
